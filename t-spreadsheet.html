Spreadsheets are very useful for storing data or making calculations. You can make and edit them with LibreOffice Calc

Pre-requisites

  • LibreOffice
  • Data that would be best viewed or calculated in/with a spreadsheet

Steps

  1. Open LibreOffice.
  2. Open the Calc program, as shown in Figure 1.
  3. If you want to edit an existing spreadsheet, select the "Open File" button instead.
  4. Type the values into the spreadsheet, using the format options shown in figure 2 to alter the spreadsheet as needed.
  5. To use equations, go to "format", then "function" as shown in figure 3. An example function is shown in figure 4.
  6. Periodically save your spreadsheet by pressing ctrl + "s" at the same time.
  7. When finished, either exit LibreOffice by clicking the X at the upper right of the screen.
