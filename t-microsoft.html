First things first: To use LibreOffice, first you have to download it.

Context

Before this guide can teach you how to use LibreOffice, you must download the program onto your computer.

"Ingredients"

  • Your computer

  • Internet Access

  • Enough time and patience to wait through the download and installation

Steps

  1. Go to http://www.libreoffice.org.
  2. On that page, click the "Download Now" button shown in figure 1.
  3. From the next page, select the download button under the "Main Installer Header." (See Figure 2)
  4. Your download should start immediately. Once the download finishes, open the installer from the download page of your browser and click "run." (See Figure 3)
  5. Follow the installation wizard's prompts.
  6. After the installation is finished, LibreOffice can be accessed like any other program from the start menu, or the desktop shortcut if you chose to have one made when prompted by the installation wizard.
