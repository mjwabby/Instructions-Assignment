Powerpoints are very useful for presentations. You can make and edit them with LibreOffice Impress

Pre-requisites

  • LibreOffice
  • A presentation idea

Steps

  1. Open LibreOffice.
  2. Open the Impress program, as shown in Figure 1.
  3. If you want to edit an existing presentation, select the "Open File" button instead.
  4. In order to add a new slide or change the format of a given slide, right click on one of the slide previews to the left, then use the resulting menu, shown in red in figure 2.
  5. Add text and other content to each slide as need be, formatting your presentation using the buttons shown in blue in figure 2.
  6. Periodically save your presentation by pressing ctrl + "s" at the same time.
  7. When finished, either exit LibreOffice by clicking the X at the upper right of the screen.
